# Personal Expense Tracker 💰

A straightforward JavaScript utility to analyze monthly expenses, filter them by category, and sort them by amount.

## 🚀 Features
- Filters transactions to isolate specific categories (e.g., "Food").
- Calculates the total money spent in that specific category using the `reduce()` method.
- Sorts the entire expense list in descending order (highest to lowest) to quickly identify major expenses.

## 🧠 Concepts Used
- Array Methods (`filter`, `reduce`, `sort`)
- Custom Compare Function for Sorting Numeric Object Properties
- Arrow Functions

## 🛠️ How to Run
Run the JavaScript file using Node.js:
